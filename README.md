# 🚀 AI Code Reviewer

An AI-powered Code Reviewer built using **React.js**, **Node.js**, **Express.js**, and the **Groq API**. The application analyzes code submitted by users and provides intelligent feedback on bugs, code quality, security, performance, syntax, and best practices.

## ✨ Features

- 🤖 AI-powered code review
- 📝 Supports reviewing code snippets
- ⚡ Fast responses using Groq Llama 3.3
- 🔒 Security suggestions
- 🚀 Performance optimization tips
- ✅ Best practices recommendations
- 📊 Code quality analysis
- 💡 Markdown formatted responses
- 🌍 Responsive UI
